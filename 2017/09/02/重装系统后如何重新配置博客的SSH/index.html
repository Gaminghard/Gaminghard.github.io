重装系统后如何重新配置博客的SSH

2017-09-02

我这里使用的是hexo+github搭建的博客。因为之前多次重装系统导致了博客无法更新（原本hexo文件夹保留），需要重新配置SSH。把重新配置SSH的方法记录下来，以防之后又遇到相同问题。

1.准备环境

1.安装git

2.下载安装node.js

3.运行git，输入npm install hexo-cli -g，安装hexo

2.生成SSH

1.运行git，输入ssh-keygen -t rsa -C “youremail@example.com” （youremail@example.com为你的邮箱地址），之后一直回车下一步。

2.在你的用户主目录下会生成一个.ssh目录。里面有id_rsa和id_rsa.pub两个文件，前者为私钥，后者为公钥。我们需要用公钥来配置博客。

（用户主目录一般为C:\Users\Administrator）

3.配置到github

登录github，进入settings中的SSH and GPG keys，点击new SSH key。随意命名title，在key中输入id_rsa.pub中的全部内容，然后Add SSH Key。

这样就完成了SSH的配置。
